# Markdown Support Guide

The AI assistant supports Markdown for rich, readable responses.

## Supported Features

### Headings

```markdown
# H1
## H2
### H3
```

### Text Formatting

```markdown
**bold**
*italic*
~~strikethrough~~
`inline code`
```

### Code Blocks

```markdown
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```
```

### Lists

```markdown
- Item 1
- Item 2
  - Sub-item
```

### Links

```markdown
[Example](https://example.com)
```

### Quotes

```markdown
> This is a quote.
```

### Tables

```markdown
| Col A | Col B |
|------|------|
| A1   | B1   |
```

## Rendering Notes

- GitHub-flavored Markdown (GFM)
- Line breaks supported
- Styled code blocks and tables
